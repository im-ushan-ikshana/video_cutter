# Contributing to Universal Video Cutter

First off, thank you for considering contributing to Universal Video Cutter! It's people like you that make open source such a great community.

## Where do I go from here?

If you've noticed a bug or have a feature request, please make sure one doesn't already exist in the Issue Tracker. If you find a matching issue, feel free to add a comment or reaction.

If your issue is new, please open a new Issue using the provided templates!

## Setting up your environment

1. Clone the repository
2. Run `./download_ffmpeg.ps1` to pull the required FFmpeg sidecars for your platform
3. Run `npm install`
4. Run `npm run tauri dev`

## Pull Request Process

1. Ensure your code aligns with SvelteKit and Tauri standards.
2. Update the `CHANGELOG.md` with details of changes to the interface or architecture.
3. Your PR will be reviewed by maintainers before merging.

Thank you!
