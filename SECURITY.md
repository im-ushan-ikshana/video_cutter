# Security Policy

## Supported Versions

Currently, only the latest release of Universal Video Cutter is officially supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Universal Video Cutter (e.g., related to the Tauri backend or FFmpeg integrations), please do **not** report it via public GitHub Issues. 

Instead, please send an email directly to the maintainers. We will acknowledge your report within 48 hours, and provide a timeline for a patch. We ask that you do not publicly disclose the issue until it has been addressed.
